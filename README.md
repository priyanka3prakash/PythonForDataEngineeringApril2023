### Git Commands

    - To create a branch,
        git chekout -b NEWBRANCHNAME
    - To check the local list of branches,
        git branch
    - To get current file/folder creation/modification changes,
        git status
    - To add(stage) the file changes,
        git add file/folder
    - To commit the staged files,
        git commit -m "commit message"
    - To push the code
        git push origin NEWBRANCHNAME

    Then, in the github UI, we need to create Pull request
